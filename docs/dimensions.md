# Dimensions

Lumen scores seven dimensions inspired by the Pathological Narcissism Inventory. These are research dimensions, not clinical categories.

| Dimension | Subtype | Plain-language description |
| --- | --- | --- |
| Grandiose Fantasy | Grandiose | Fantasies of admiration, success, or superiority |
| Self-Sacrificing Self-Enhancement | Grandiose | Prosocial behavior used to support a positive self-image |
| Exploitativeness | Grandiose | Using others for personal advantage |
| Entitlement Rage | Vulnerable | Anger when expected recognition or special treatment is denied |
| Contingent Self-Esteem | Vulnerable | Self-worth that depends heavily on approval or validation |
| Hiding the Self | Vulnerable | Concealing needs, flaws, or vulnerability |
| Devaluing | Vulnerable | Dismissing or devaluing others after disappointment |

Grandiose dimensions emphasize admiration, superiority, image, or interpersonal advantage. Vulnerable dimensions emphasize shame, contingent self-worth, concealed need, anger around recognition, and devaluation after disappointment.

These descriptions are simplified for public interpretation. The scoring procedure uses structured rubrics rather than these short descriptions alone.
